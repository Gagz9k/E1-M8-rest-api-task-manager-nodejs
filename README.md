# ✅ Task Manager API

REST API for task management built with **Node.js**, **Express**, **Sequelize**, and **PostgreSQL**.

This project implements a complete CRUD system for managing tasks, following RESTful principles and proper HTTP status codes.

---

## 🚀 Features

- Create tasks
- Read all tasks
- Read a task by ID
- Update tasks
- Delete tasks
- JSON request handling
- Sequelize ORM integration
- PostgreSQL database connection
- Basic validation and error handling

---

## 🛠 Tech Stack

- Node.js
- Express
- Sequelize
- PostgreSQL
